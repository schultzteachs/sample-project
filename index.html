Tutorial for Git and Github

Commands for local Git

git init

git status

git add .

git commit -m "message"

Commands for Github

git remote add origin [repository URL]

git push -u origin main

Set up user information for Git

You might get the error: "src refspec main does not match any". In this case, you need to use git config to set your user information.

git config --global user.name "Your Name"

git config --global user.email "you@example.com"

Then try committing and pushing again.

git push (pushes local commits to Github)

Workflow generally is:

git add . (stages all changes)

git commit -m "message" (commits staged changes with a message)

git push (pushes local commits to Github)

git status (shows the status of changes)

You should get something that looks like this:

$ git status On branch main Your branch is up to date with 'origin/main'. nothing to commit, working tree clean
